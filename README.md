# minion_chatbot
nodejs webapp leveraging openai's chatgpt model

## Setup

`git clone https://github.com/technomystics-org/minion_chatbot`

`cd minion_chatbot`

Modify `.env.example` and add your OpenAI API Key.

Save as `.env`

## Run

`node app.js`

Browse to `localhost:5000`
